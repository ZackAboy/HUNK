# Hunk

Hunk is a Flutter-based cross-platform AI fitness coach app for iOS and Android.

The current app is an initial shell with four placeholder screens:

- Home
- Health
- Coach
- Settings

Read `docs/AGENT.md` before making project changes. The main project context lives in `docs/`.

## Run

```sh
flutter pub get
flutter run
```

## Test

```sh
flutter analyze
flutter test
```
